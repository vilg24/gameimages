# Attack Animations

This folder is generated by `tools/generate_attack_animations.py`.

## Source inputs

The generator reads these source PNG files:

- `assets/source/lightning_anime_source.png`
- `assets/source/flaming_sword_source.png`

## Generate outputs

From the repository root:

```bash
python3 tools/generate_attack_animations.py
```

The script is deterministic (fixed random seed), overwrites outputs safely, and generates:

- `assets/animations/lightning_attack/`
- `assets/animations/flaming_sword_slash/`

Each effect includes:

- `frames/*.png` (10 frames, 512x512 RGBA transparent)
- `spritesheet/*_sheet.png` (5 columns x 2 rows)
- `*.json` (frame order, frame rects, fps, pivot)
- `preview/*.gif` (quick preview)

## Metadata JSON schema

Example structure:

```json
{
  "name": "lightning_attack",
  "fps": 12,
  "frame_size": [512, 512],
  "frames": [
    {"file": "frames/lightning_attack_000.png", "index": 0, "rect": [0, 0, 512, 512]}
  ],
  "sheet": {"file": "spritesheet/lightning_attack_sheet.png", "cols": 5, "rows": 2},
  "pivot": [0.5, 0.5]
}
```
